# TextEditor  
==============

A simple Windows Notepad-like text editor built using C# and Windows Forms in Visual Studio.  
This project demonstrates core concepts of desktop application development, including file handling, text editing, formatting, dialogs, and document printing.  

The application allows users to create, edit, open, format, and print text documents using an interface similar to the classic Windows Notepad.  

## Features  
===========  

### File Operations  

. Create New document  
. Open existing text files  
. Save documents  
. Print documents  
. Print Preview  
. Exit application  

### Editing Tools

. Undo  
. Cut  
. Copy  
. Paste  
. Select All  
. Find text in document  

### Text Formatting
. Change Font  
. Change Font Color  
. Change Background Color  
. Text Alignment
